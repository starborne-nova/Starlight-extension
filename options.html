Recent News
  • The Data Repair and Reset options should fix any of the settings acting funny. I changed the way data is structured and extensions are pretty stupid with the way they handle storage in general
  • The popup now has pages to reduce clutter. A future option will let you set the number of items per page
  • Almost all of the things are in place to start twitch user integration. You will not need to authorize starlight, just enter your username to search follows. To manage server stability, you will be limited to the amount of "unfollowed by server" streamers you can add at first. This limit will be raised depending on a variety of factors.
  • At this point, a small push is being made to expand the userbase to load test the server behind Starlight. If you have suggestions of smaller streaming communities who might like to be added, or if you notice significant delays in twitch data (Greater than 10 minutes), please send an email. Sometimes twitch resets the token and data will be unavailable for a few hours. This is being addressed.
  • Thank you for using Starlight! The next stage of the project is approaching and any large changes to settings will be documented on the options page.
